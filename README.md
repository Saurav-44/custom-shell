# myshell — Custom Shell Implementation (Capstone Assignment 2)

## Build
```
make
```

## Run
```
./myshell
```

## Features
- Run external programs
- Built-in commands: cd, exit
- Background execution using `&`

## Example
```
ls -l
sleep 10 &
cd /tmp
exit
```
